Welcome

This project aims to complete the Supyire-French-English dictionaries, a mobile app, and a digital text corpus. Supyire is spoken by 350,000+ people in Mali and this work will support education, cultural preservation, and research.

About the Project

The Supyire Dictionary and Text Database Project was initiated by Prof. Robert Carlson, with contributions from Robert Sanogo and Michael Otieno. It includes the development of dictionaries, interlinearized texts, and language learning tools for Supyire, a national language in Mali.
